Siphon

Siphon is a small platformer written in C using SDL and OpenGL. It currently has five small levels that consist of some small puzzles with the objective being to get to the end of the level. The puzzles involve a rope with two separate ends that can be picked up. These can be thrown onto high ledges after which the rope can be retracted in order to get yourself up that ledge.

Controls

  • escape/enter/arrow keys, navigate menu
  • arrow keys, to move
  • a, jump
  • d, pick up rope
  • s, retract rope
  • f, throw rope
  • f+up, throw rope upwards
  • f+down, put down rope
  • down, pick up powerup
  • n, select next player
  • p, pause the game
  • r, rewind the game (useful if you make a mistake)
